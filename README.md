# üå± fakeGreen


fakeGreen is a fun project aimed at demonstrating how GitHub's contribution graph can appear more "green" (i.e., active) through automated commits. This repository offers a simple way to generate daily commits, making your contribution graph stand out.

> **Important:**  
> To use this project, you **must first create your own private repository**.  
> **Do NOT use or fork this public repository directly.**  
> 1. Create a new private repository on your GitHub account.
> 2. Copy all files from `fakeGreen` to your private repository.
> 3. Commit and push the files to your own repository.
> This ensures your activity remains private and unique to you.

## Features

- Daily commits for a "greener" contribution graph
- Simple to set up and customize
- Lightweight and minimal dependencies
- The commits are random so they may apear legit
- For automated daily commits you can run it on a server and add a sleep timer for the command

## Usage

1. **Create your own private repository:**  
   - Go to GitHub and click "New repository".
   - Set the visibility to **Private**.

2. **Copy this project's code:**  
   - Download or clone the `fakeGreen` repository.
   - Copy all files to your own private repo.

3. **Commit and push:**  
   ```bash
   git add .
   git commit -m "Setup fakeGreen"
   git push origin main
   ```

4. **Use it:**  
   ```bash
   python3 main.py
   ```
